# Portfolio

Personal portfolio website built as a simple static project to present basic information, skills, and projects.

## Live Demo
https://liakops7.github.io/cv/

---

## Overview

This repository contains a **static portfolio website** created using plain web technologies.
The goal of the project is to provide a clean and lightweight personal portfolio without frameworks or build tools.

---

## Features

- Single-page layout
- Responsive design
- Sections for personal information and projects
- Simple and fast loading (no frameworks)

---

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)
